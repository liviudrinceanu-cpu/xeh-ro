## DATE TEHNICE NECESARE PER PRODUS

Pentru fiecare produs vom prelua:

1. **Identificare**
   - Cod produs (SKU)
   - Model
   - Brand (RM / REDFOX)
   - Linie (600 / 700 / 900)

2. **Descriere**
   - Denumire RO (SEO-optimizată)
   - Descriere scurtă
   - Descriere completă

3. **Specificații Tehnice**
   - Dimensiuni (L x l x h) mm
   - Greutate (kg)
   - Alimentare (V / Hz / Phase)
   - Putere electrică (W/kW)
   - Putere gaz (kW)
   - Tip gaz (natural / GPL)
   - Capacitate (litri / GN)
   - Temperatură lucru (°C)
   - Material construcție

4. **Media**
   - Imagine principală
   - Imagini galerie
   - Fișe tehnice PDF
   - Certificate

5. **Prețuri**
   - Preț catalog EUR
   - Preț RON (curs BNR)

6. **Stare**
   - Disponibilitate
   - Termen livrare
   - Noutate / Lichidare

Prețurile de pe site-urile sursă sunt B2B (pentru dealeri). 
Va trebui să decizi:
- Afișezi prețuri? (da/nu)
- Ce tip de prețuri? (catalog / dealer / retail)
- Curs valutar EUR→RON

- RM: ~500-800 produse
- REDFOX: ~800-1200 produse
